# Wikid
이 프로젝트는 지인들의 위키를 직접 작성하고 공유하는 플랫폼입니다.

### 팀원 소개 및 역할 분담

- **강희정**: 내위키 페이지, 위키페이지(콘텐츠 있음)
- **박하은**: 랜딩페이지, 공통 컴포넌트, 모달
- **손혁진**: 회원가입, 로그인, 계정설정
- **이윤승**: 게시글 상세페이지, 게시물 등
- **전수영**: 위키 목록 페이지, 자유게시판
